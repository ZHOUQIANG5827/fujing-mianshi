# 辅警面试真题卡组

深圳辅警面试历年真题卡组，含年份/题型/标签筛选、随机抽题模拟考场、录音演练与语音转文字预览。

## 本地预览
直接双击打开 `index.html` 即可（部分功能如麦克风录音需要 HTTPS 环境，本地 file:// 打开会受限）。

## 部署
本项目已连接 Cloudflare Pages，推送到 main 分支会自动触发发布。

## 页面说明
- `index.html`：面试真题卡组（138 题，年份/题型/标签筛选，录音演练 + 语音转文字预览）
- `written.html`：笔试刷题库（900 题，单选/多选/判断，点选即判分并展示解析，随机刷题模式）

两个页面互相有顶部跳转入口，都是完全自包含的单文件（无需额外依赖），直接放进同一个仓库根目录部署即可。
